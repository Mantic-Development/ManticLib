# ManticLib
Libraries for any mantic plugin


Repository:
```xml 
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository> 
  ```

Dependency:
```xml
<dependency>
    <groupId>com.github.Mantic-Development</groupId>
    <artifactId>ManticLib</artifactId>
    <version>(version)</version>
    <scope>provided</scope>
</dependency>
  ```
